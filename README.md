# Hello Wallpaper

A macOS menu bar app for managing desktop wallpapers with automatic Light/Dark mode switching.
<img width="1800" height="1104" alt="screenshot-20260106-155154" src="https://github.com/user-attachments/assets/8fd5c5fc-aac7-4dff-96a7-4b3d5d4f68c4" />

## Features

- **Wallhaven Integration**: Search and download high-quality wallpapers from [Wallhaven.cc](https://wallhaven.cc)
- **Local Files**: Import wallpapers from your computer
- **Light/Dark Mode**: Set different wallpapers for Light and Dark modes
- **Auto Switch**: Automatically change wallpaper when system appearance changes
- **Favorites**: Save your favorite wallpapers for quick access
- **Menu Bar**: Quick access from the system tray

## Requirements

- macOS 14.0+
- Xcode 15.0+

## Usage

1. Click the photo icon in the menu bar
2. Open the main window from the menu
3. Browse wallpapers:
   - **Explore**: Search Wallhaven for wallpapers
   - **Local**: Add wallpapers from your computer
   - **Favorites**: View saved wallpapers
   - **Current**: Manage Light/Dark mode wallpapers
4. Click a wallpaper to preview and set it
